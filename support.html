Getting started

  1. 1 Open Stark and click Add Folder in the sidebar to grant access to a folder.
  2. 2 Select the folder you want to organize from the file browser.
  3. 3 Type a command in the AI panel — for example "organize by date" or "sort by file type".
  4. 4 Review the proposal, approve the moves you want, then click Apply.
  5. 5 If anything looks wrong, open History and hit Undo.

Common questions

Stark says it can't access my folder

Stark requires explicit permission for each folder. Go to Sidebar → Add Folder and select the folder in the file picker. macOS will ask you to confirm access. If you previously denied access, go to System Settings → Privacy & Security → Files and Folders and re-enable Stark.

The AI command isn't working or gives unexpected results

Try rephrasing your command. Simple, direct instructions work best — e.g. "group by file type" or "move PDFs to Documents". If the AI pipeline isn't available (requires macOS 26+), Stark falls back to a built-in rule-based organizer. Make sure you're on macOS 26 or later for the full AI experience.

How do I undo an organization?

Open the History tab from the toolbar. Every session is listed with a timestamp. Click Undo next to any batch to reverse it, or use Undo to Here to roll back multiple sessions at once.

Will Stark touch my Git repos or Xcode projects?

No. Stark automatically detects and protects folders containing .git, node_modules, .xcodeproj, Pods, Cargo.toml, package.json, and other project markers. These are excluded from all operations by default.

I bought Stark Pro but it's still showing as Free

Open Stark → go to Settings → tap Restore Purchases. This syncs your App Store purchase with the app. If the issue persists, make sure you're signed into the same Apple ID used for the purchase and contact us at ata@kilicov.dev.

How do I set up automatic rules?

Click the Rules button in the toolbar to open the Rules window. Tap + to add a new rule — choose a trigger (file type, name pattern, or AI content match) and a destination folder. Rules run automatically when you organize a folder. Free users can create 1 rule; Stark Pro unlocks unlimited rules.

Can I use Stark in Turkish?

Yes. Stark supports Turkish commands — for example "tarihe göre grupla" or "fotoğrafları ayır". You can also change the app language to Turkish in Settings → Language.

How do I report a bug?

Email us at ata@kilicov.dev with a description of what happened, your macOS version, and any steps to reproduce the issue. Screenshots are always helpful.
